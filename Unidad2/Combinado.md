# **Ejercicio Combinado en Java**

Este programa solicita al usuario el numero de estudiantes a los que desea calcular el promedio general y, a partir de ese número, utiliza un ciclo for para repetir el proceso de ingreso y validación de notas. En cada iteración, el programa muestra el número del estudiante y luego emplea un ciclo do–while para pedir su calificación, asegurándose de que el valor ingresado sea válido, es decir, que esté dentro del rango de 0 a 10. Si la nota no cumple con este requisito, el programa informa que es incorrecta y solicita que se ingrese nuevamente. Cuando se obtiene una nota válida, esta se suma a la variable acumuladora que registra el total de notas ingresadas. Finalmente, después de procesar a todos los estudiantes, el programa calcula el promedio dividiendo la suma acumulada entre la cantidad total de estudiantes y muestra el resultado en pantalla, indicando el promedio final de sus calificaciones.
