# If **Simple**

Una condición IF es una estructura de programación que permite que un programa tome decisiones según una condición lógica. 
Esta condición se evalúa como verdadera o falsa, de esa manera es cómo se define si un bloque de código debe ejecutarse o no. 
El IF controla el flujo del programa, guiando qué acciones ocurren y cuáles se omiten, lo que lo convierte en una herramienta esencial
para crear comportamientos dinámicos y respuestas inteligentes dentro del software[6].

A continuacion se muestra un ejemplo del If simple, en etse ejercicio solicita al usuario que ingrese su edad y luego utiliza una condición if para verificar si el valor ingresado es mayor o igual a 18. Si esta condición se cumple, el programa muestra el mensaje "Eres mayor de edad", indicando que la persona tiene la edad suficiente para ser considerada adulta legalmente. En caso de que la edad sea menor a 18, el programa no muestra ningún mensaje adicional, ya que se trata de un if simple sin bloque else. En resumen, el ejercicio evalúa una edad ingresada y muestra un mensaje únicamente cuando la persona cumple con la mayoría de edad.

# **if – else**
El IF–ELSE es una estructura de programación que permite que un programa elija entre dos caminos posibles según una condición lógica. 
Primero se evalúa una condición y si resulta verdadera, entonces se ejecuta un primer bloque de instrucciones, y si es falsa, se ejecuta 
un bloque alternativo. Esta estructura permite que el programa respondade manera diferente según la situación, haciendo que su comportamiento sea más flexible y dinámico[7].

A continuacion se muestra un programa el cual solicita al usuario que ingrese una calificación y luego utiliza una estructura condicional if–else para evaluar ese valor. Si la nota ingresada es mayor o igual a 10, el programa muestra el mensaje "Aprobado", indicando que el estudiante cumple con el puntaje mínimo necesario. En caso contrario, es decir, si la nota es menor a 7, se ejecuta el bloque else y se muestra el mensaje "Reprobado". En resumen, el programa determina si un estudiante aprueba o reprueba según la calificación que ingresa.
