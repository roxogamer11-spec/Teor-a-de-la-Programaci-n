# **if simple**

Una condición IF es una estructura de programación que permite que un programa tome decisiones según una condición lógica. 
Esta condición se evalúa como verdadera o falsa, de esa manera es cómo se define si un bloque de código debe ejecutarse o no. 
El IF controla el flujo del programa, guiando qué acciones ocurren y cuáles se omiten, lo que lo convierte en una herramienta esencial
para crear comportamientos dinámicos y respuestas inteligentes dentro del software[6].

A continuacion se muestra un ejemplo del If simple, en etse ejercicio solicita al usuario que ingrese su edad y luego utiliza una condición if para verificar si el valor ingresado es mayor o igual a 18. Si esta condición se cumple, el programa muestra el mensaje "Eres mayor de edad", indicando que la persona tiene la edad suficiente para ser considerada adulta legalmente. En caso de que la edad sea menor a 18, el programa no muestra ningún mensaje adicional, ya que se trata de un if simple sin bloque else. En resumen, el ejercicio evalúa una edad ingresada y muestra un mensaje únicamente cuando la persona cumple con la mayoría de edad.

# **if – else**
El IF–ELSE es una estructura de programación que permite que un programa elija entre dos caminos posibles según una condición lógica. 
Primero se evalúa una condición y si resulta verdadera, entonces se ejecuta un primer bloque de instrucciones, y si es falsa, se ejecuta 
un bloque alternativo. Esta estructura permite que el programa respondade manera diferente según la situación, haciendo que su comportamiento sea más flexible y dinámico[7].

A continuacion se muestra un programa el cual solicita al usuario que ingrese una calificación y luego utiliza una estructura condicional if–else para evaluar ese valor. Si la nota ingresada es mayor o igual a 7, el programa muestra el mensaje "Aprobado", indicando que el estudiante cumple con el puntaje mínimo necesario. En caso contrario, es decir, si la nota es menor a 7, se ejecuta el bloque else y se muestra el mensaje "Reprobado". En resumen, el programa determina si un estudiante aprueba o reprueba según la calificación que ingresa.

# **if - else - if**

El condicional IF-ELSEIF es una estructura de control que permite evaluar múltiples condiciones de forma secuencial. Primero se debe verificar una condición con if; si esa no se cumple, el programa pasa a evaluar otra con else if, y así sucesivamente. Al final, si ninguna de las condiciones anterioreses verdadera, se puede usar un else para ejecutar un bloque por defecto. Esta sintaxis es más legible y cómoda que anidar múltiples if, ya que evita estructuras complejas y facilita seguir el flujo lógico de las decisiones. [8]

A continuacion se muestra el ejemplo de un programa el cual pide al usuario que ingrese su edad y, mediante una estructura condicional if – else if – else, clasifica a la persona dentro de una categoría según el rango en el que se encuentre. Primero verifica si la edad es menor o igual a 12 para asignar la categoría “Niño”; si no coincide, revisa si es menor o igual a 17 y muestra “Adolescente”; si tampoco cumple, evalúa si la edad es menor o igual a 64 para mostrar “Adulto”. Finalmente, si ninguna de las condiciones anteriores se cumple, el programa asigna la categoría “Adulto mayor”. En resumen, el programa determina en qué etapa de la vida se encuentra una persona según la edad que ingresa.
