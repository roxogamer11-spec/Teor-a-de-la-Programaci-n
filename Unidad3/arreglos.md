# **Arreglos**

Un arreglo es una forma de guardar varios datos del mismo tipo usando un solo nombre. En lugar de crear muchas variables, el arreglo las organiza en una lista donde cada dato tiene una posición o índice, que normalmente empieza desde 0. Gracias a esto, es fácil acceder a cualquier dato del arreglo cuando se lo necesita, lo que hace que los arreglos sean muy útiles para trabajar con muchos valores de manera ordenada[18]. 

Una de las principales ventajas de los arreglos es que permiten acceder rápidamente a sus elementos, ya que solo se usa el índice para obtener un valor específico. Además, son fáciles de usar y ayudan a mantener el código más ordenado. Sin embargo, también tienen una limitación importante: su tamaño se define desde el inicio y no puede cambiar, lo que puede ser un problema si no se sabe cuántos datos se van a necesitar[19].

Además de los arreglos simples, existen los arreglos multidimensionales, como las matrices, que se usan para representar información en forma de tablas, filas y columnas. Estos arreglos son muy comunes en programación, ya que sirven como base para realizar operaciones como recorridos con bucles, búsquedas y cálculos, siendo fundamentales en muchos programas y estructuras más avanzadas[20].

# **Ejemplos de Arreglos**

## **Arreglo unidimensional**

Un arreglo unidimensional es una estructura que almacena varios elementos del mismo tipo de datos en una secuencia lineal, uno después de otro en memoria. Cada elemento se identifica por un índice único que empieza desde 0, y permite acceder o modificar el dato fácilmente sin necesidad de muchas variables separadas[21].

## **Arreglo bidimensional**

Un arreglo bidimensional es básicamente una matriz o tabla de datos, donde los elementos se organizan en filas y columnas. En lenguajes como C, se declara con dos dimensiones, y cada valor se accede usando dos índices, uno para la fila y otro para la columna, lo que lo hace útil para representar datos tabulares como tablas de notas, gráficos o mapas[22].

## **Arreglo tridimensional**

Un arreglo tridimensional es un arreglo que contiene varias matrices dentro de sí, permitiendo representar estructuras de datos más complejas, como un conjunto de planos o capas de datos. En C se declara con tres dimensiones, donde cada índice representa una posición en cada nivel como plano, fila y columna, lo que facilita organizar colecciones de matrices relacionadas[23].
